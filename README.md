# Hardin Trading Software v2

A fresh build — new Alpaca account, new strategy, new approach.

## Setup

1. Create a new Alpaca paper trading account at https://alpaca.markets
2. Copy `.env.example` to `.env` and fill in your new API keys
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python app.py`
